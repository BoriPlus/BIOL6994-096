1) Versin 1:

Se limpi el fondo y se aadieron ejes mas claros editados directamente con ggplot. Se utiliz theme_bw() para hacer un fondo blanco limpio contrario a la original que era gris. Se utiliz labs() para poner ttulos informativos de ejes y un ttulo general, contrario a la original que fueron los ejes directos del RAW data. Por ltimo, se utiliz theme() para aumentar el tamao de las letras.

ggplot(metabric, aes(x = type_of_breast_surgery, y = tumor_size)) +
         geom_boxplot() +
        # Etiquetas de los ejes y el ttulo del grfico
         labs(x = "Type of breast surgery",
              y = "Tumor Size (mm)",
              title = "Tumor Size Distribution by Type of Surgery"
              ) +
        # Tema de fondo blanco
         theme_bw() +
        # Aspectos estticos del grfico (ttulo, ejes y texto)
         theme(
           plot.title = element_text(hjust = 0.5, size = 16, face = "bold"),
           axis.title = element_text(size = 12, face = "bold"),
           axis.text  = element_text(size = 12, face = "bold")
         )

2) Versin 2:

Se aadieron colores a las cajas de distribucin de datos, se mejor las etiquetas para que todo estuviera en ingls y se aadi una leyenda con los colores de las cajas. Se utiliz fill = type_of_breast_surgery para diferenciar visualmente cada grupo con una comparacin ms rpida y se dejaron con colores automticos que dej ggplot. Se utiliz scale_x_discrete(labels = c()) para cambiar las etiquetas de categoras a ortografia correcta sin todo mayuscula. Se trat de cambiar los datos de NA a Non Reported. La libreria ggplot2 lo exhibe como NA pero no puede etiquetar NA como una palabra, porque lo borra del eje tratandolo como un dato faltante (NULL) y no como dato categrico. Se utiliz theme_minimal() para dejar un diseo sencillo sin el efecto de borde alrededor.

ggplot(metabric, aes(x = type_of_breast_surgery, y = tumor_size, fill = type_of_breast_surgery)) +
  # Boxplot y show.legend = TRUE para mantener la leyenda visible
  geom_boxplot(show.legend = TRUE) +
  # Etiquetas del grfico y de la leyenda
  labs(
    fill = "Type of Breast Surgery",
    x = "Type of Breast Surgery",
    y = "Tumor Size (mm)"
  ) +
  # Cambiamos las etiquetas de las categoras del eje X
  scale_x_discrete(
    labels = c(
      "BREAST CONSERVING" = "Breast Conserving",
      "MASTECTOMY"        = "Mastectomy",
      "NA"                = "Non Reported"
    )
  ) +
  # Tema minimalista para estilo limpio
  theme_minimal()

3) Versin 3:

Aqui se mantuvo el boxplot clsico tradicional, mejorando la interpretacin del grfico aadiendo la media de cada grupo mediante stat_summary(), representado con un punto de color negro para facilitar la comparacin visual entre promedios. Se utiliz coord_cartesian(ylim = c(0, 175)) para ajustar manualmente la escala del eje Y sin modificar los datos ni quitar de vista los valores extremos outlayers y que sigan formando parte del anlisis. Aqu, en vez de leyenda, como version 3 se realiz un resumen estadstico dentro del propio grfico, utilizando annotate(). Este comando genera un cuadro de leyenda tipo texto con la media, mediana y tamao de muestra (n) calculados directamente desde el conjunto de datos de la tabla brca_metabric_clinical_data.csv. Esto permite al lector conocer alguna informacin de estadistica descriptiva relevante sin necesidad de generar tablas por separado o ir directamente al archivo RAW a ver los datos y realizar dichos calculos aparte. Finalmente, se us theme_classic() para obtener un diseo clsico tradicional con ejes delimitados.

ggplot(metabric, aes(x = type_of_breast_surgery, y = tumor_size)) +
  geom_boxplot() +
  # Punto negro que marca la media para cada categora
  stat_summary() +
  # Ajuste del rango de escala del eje Y sin alterar los datos
  coord_cartesian(ylim = c(0, 175)) +
  # Cambio de etiquetas del boxplot
  labs(
    x = "Type of Breast Surgery",
    y = "Tumor Size (mm)"
  ) +
  # Tema clsico con bordes en los ejes
  theme_classic() +
  # Insercin de resumen estadstico dentro del grfico
  annotate("text",
           x = 2.5, 
           y = 160,
           hjust = 0,
           size = 4,
           label = paste(" Summary Statistics\n",
             "for Tumor Size (mm)\n",
             # Utilizazion de round para redondear a 2, 
             # na.rm para que cuente como dato los valores vacios y no de error, 
             # !is.na para trasnformar aquellos valores vacios en 0 y que ejecute el comando sumando los
             # valores de la tamao tumoral, osea, cuenta cuntos valores vlidos hay
             "Mean: ", round(mean(metabric$tumor_size, na.rm = TRUE), 2), "\n",
             "Median: ", round(median(metabric$tumor_size, na.rm = TRUE), 2), "\n",
             "n: ", sum(!is.na(metabric$tumor_size))
           )
  )
